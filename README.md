# Margic

## 🎓 Story Behind the Project

I built this project on a deadline that I only realized the night before the competition. The idea was simple: give students access to free career counselling through an AI-powered helper. I wanted to make something useful for other students, especially those who don’t have easy access to guidance.

It started as a last-minute sprint, built in one night with Streamlit and Google Gemini. I barely knew the competition was the next day, but I still managed to get a working prototype together, refine the presentation, and deliver something creative.

## 🚀 What It Does

`margic.py` is a student-friendly career path finder that:
- collects academic and personal background information,
- stores a profile,
- uses an AI chatbot to offer practical career advice,
- suggests direction, identifies gaps, and recommends what to improve next.

The goal is to make career coaching feel real, honest, and motivating.

## 🧠 Built With

- Python
- Streamlit for the UI
- Gemini AI for the chatbot
- JSON for saving student profiles

## 🏆 Outcome

This project ended up winning 1st place in the interschool competition. The presentation was one of the strongest parts of the submission — creative, polished, and aligned with the story of a student building something meaningful at the last minute.
